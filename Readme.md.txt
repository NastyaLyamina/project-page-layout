# Вёрстка главной страницы проекта «Тишинский бульвар»

Адаптивная верстка современного лендинга с акцентом на чистоту кода и соответствие макету.

[https://nastyalyamina.github.io/project-page-layout/]

## 🛠 Технологии и инструменты
* HTML5 — семантическая разметка для SEO и доступности.
* Sass - стилизация выполнена с помощью препроцессора.
* Flexbox/Grid - современная верстка без лишних библиотек.
* БЭМ (Block Element Modifier) — методология именования классов для поддержки кода.
* Adaptive Design — корректное отображение на устройствах от 320px до 1920px.
* Pixel Perfect — строгое соответствие исходному макету из Figma.

## 🚀 Особенности проекта
* Кроссбраузерная верстка (Chrome, Firefox, Safari).
* Оптимизированные изображения для быстрой загрузки.
* Интерактивные элементы: плавный скролл, анимации при наведении (hover-эффекты).

## ⚙️ Как запустить локально
1. Склонируйте репозиторий:
   `git clone https://github.com`
2. Откройте файл `index.html` в вашем браузере.
